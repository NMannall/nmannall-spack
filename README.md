# nmannall-spack
Repo to store my spack repositories and environments

## Load spack

### ARCHER2

```
module load other-software
module load spack
```

### Cirrus

```
module laod spack
```

## Add repository

From the root of this repository:

```
spack repo add spack_repo/[repo-name]
```

## Add environment

From the root of this repository:

```
spack env track environments/[environment-name]
```

When using software installed with spack, I would recommend using the module files generated by Spack.
